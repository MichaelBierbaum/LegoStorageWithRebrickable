# LEGO Search

LEGO Search app using Angular, RxJS, TypeScript, and Rebrickable API.

## Getting Started

1. Clone the repo  
  `git clone https://github.com/nightwolfdev/lego.git`
2. Navigate to the project folder  
  `cd lego`
3. Install dependencies  
  `npm install`
4. Run the app  
  `ng serve`
5. Visit the app in a browser  
  `http://localhost:4200`
